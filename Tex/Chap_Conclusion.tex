\section{全文总结}

本文针对多模光纤散斑图像重建问题，系统地研究了基于深度学习的图像重建方法，主要完成了以下工作：

% TODO: 总结各章的主要工作
\begin{enumerate}
    \item 第一章系统综述了多模光纤成像技术的研究背景、发展现状及面临的挑战，明确了本文的研究目标和内容安排。
    \item 第二章详细阐述了多模光纤成像的理论基础，包括光纤传输理论、散斑图案形成机制以及深度学习在图像重建中的应用原理。
    \item 第三章基于公开数据集开展了初步研究，验证了深度学习方法在多模光纤散斑图像重建中的可行性和有效性。
    \item 第四章提出了基于双编码方案的多模光纤散斑图像重建方法，设计并实现了DeepLeakyU-Net网络架构，通过实验验证了所提方法的优越性能。
\end{enumerate}


\section{主要创新点}

本文的主要创新点包括：

% TODO: 列出论文的创新点
\begin{enumerate}
    \item 提出了全息编码与偏振编码相结合的双模态编码方案，显著提升了多模光纤成像的信息容量和重建质量。
    \item 设计了DeepLeakyU-Net深度神经网络架构，通过Leaky ReLU激活函数改善了深层网络的训练效果，提高了图像重建的准确性。
    \item 搭建了完整的双编码光学实验系统，构建了大规模数据集，系统验证了所提方法在实际应用中的有效性和鲁棒性。
\end{enumerate}


\section{研究展望}

尽管本文在多模光纤散斑图像重建方面取得了一定的研究成果，但仍存在一些问题有待进一步研究：

% TODO: 列出未来的研究方向
\begin{enumerate}
    \item \textbf{提高重建分辨率}：进一步优化网络结构和训练策略，探索更高分辨率图像的重建方法，以满足高精度成像应用的需求。
    \item \textbf{增强系统鲁棒性}：研究更鲁棒的编码方案和网络架构，提高系统对光纤扰动、环境变化等因素的抵抗能力，实现稳定可靠的长期成像。
    \item \textbf{实现实时成像}：优化算法复杂度，探索模型压缩和加速技术，开发实时图像重建系统，拓展多模光纤成像在动态场景中的应用。
    \item \textbf{拓展应用场景}：将所提方法应用于生物医学内窥成像、光纤通信、光纤传感等更多实际场景，推动多模光纤成像技术的产业化进程。
\end{enumerate}

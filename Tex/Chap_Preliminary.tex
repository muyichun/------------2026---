本章围绕公开的“散斑—目标图像”配对数据集开展前期研究工作：以文献\citep{sun2024ol_calibrationfree}公开的数据集为基础，建立U-Net基线，并在此基础上改进得到Deep Leaky U-Net（DLU-Net）网络结构，对公开数据集进行重建实验与验证分析。本章的目的在于（1）明确公开数据集上的可复现实验流程与评价指标；（2）通过基线实验得到若干结论性观察，为第\ref{chap:method}章的实验系统与方法设计提供依据与动机。

\section{公开数据集与任务定义}\label{sec:public_dataset}

\subsection{数据集来源}\label{sec:dataset_source}

本研究第三章主要参考文献\citep{sun2024ol_calibrationfree}末尾公开的数据集，该工作面向多芯光纤（Multi-core Fiber, MCF）定量相位成像任务，构建了端到端深度学习的“散斑到相位”重建范式，并公开了两份实验数据集：手写数字数据集与服装数据集及其对应的光学散斑图像\citep{sun2023_figshare_dataset1,sun2023_figshare_dataset2}。

从本论文的角度看，这两份公开数据集为“散斑图像重建”提供了良好的验证平台：输入为检测端采集的散斑强度图，输出为对应的目标图像（ground truth，本文用“目标图像”统称其相位/强度表征），可将问题统一为监督学习的图像到图像映射任务。

\subsection{任务定义与符号约定}\label{sec:task_definition}

设散斑图像为$S\in\mathbb{R}^{H\times W}$，目标图像为$Y\in\mathbb{R}^{H\times W}$，希望学习一个可微映射$f_{\theta}(\cdot)$使得：
\begin{equation}
\hat{Y} = f_{\theta}(S),
\end{equation}
其中$\theta$为网络参数，$\hat{Y}$为重建结果。训练阶段通过最小化损失函数$\mathcal{L}(\hat{Y}, Y)$对$\theta$进行优化；测试阶段对未见样本进行推理并评估重建质量与推理速度。

\subsection{数据集特征与划分}\label{sec:dataset_stats}

文献\citep{sun2024ol_calibrationfree}报告其公开数据集包含大量成对样本（散斑、目标图像），并包含两类目标内容：手写数字与服装图像。为了便于复现，本章采用与其一致或近似的训练/验证/测试划分策略：
\begin{itemize}
    \item \textbf{训练集}：用于网络参数学习。
    \item \textbf{验证集}：用于选择超参数与观察收敛情况。
    \item \textbf{测试集}：用于最终性能评估与可视化展示。
\end{itemize}

由于数据集在不同版本或下载方式下可能存在轻微差异，本论文在实现时以“实际下载后的样本数”为准，并在实验记录中固定版本号与划分随机种子（详见第\ref{chap:method}章的工程化实现与复现实验设置）。

\subsection{数据预处理}\label{sec:dataset_preprocess}

为提升训练稳定性与收敛速度，本章对输入散斑与目标图像采用一致的数据预处理流程：
\begin{enumerate}
    \item \textbf{尺寸统一}：将输入与输出统一到网络所需的固定尺寸（$H\times W$），必要时使用中心裁剪/缩放/零填充。
    \item \textbf{强度归一化}：将散斑图像与目标图像归一化到$[0,1]$或$[-1,1]$区间，并记录归一化策略以保证可复现。
    \item \textbf{数据增强（可选）}：对训练集进行随机翻转、旋转、噪声扰动等增强，以提升泛化能力（是否启用与增强参数在实验记录中明确给出）。
\end{enumerate}

\section{基线模型：U-Net}\label{sec:baseline_unet}

\subsection{选择U-Net作为基线的原因}\label{sec:why_unet}

U-Net由于其编码器-解码器结构与跳跃连接机制，能够兼顾全局语义与局部细节，在各类图像重建任务中表现稳定（第\ref{chap:theory}章已给出其结构与优势）。因此，本章以U-Net作为公开数据集的基线模型，用于建立可对比、可复现的重建性能参考。

\subsection{网络结构与实现细节}\label{sec:unet_detail}

本章U-Net基线遵循经典结构：若干级下采样编码器与对称上采样解码器，并在对应尺度通过跳跃连接拼接特征。为便于后续对比，除非特别说明，本章所有模型使用相同的输入输出尺寸、训练轮数、批大小以及学习率调度策略。

\section{改进模型：DLU-Net}\label{sec:dlunet}

\subsection{DLU-Net的设计动机}\label{sec:dlunet_motivation}

在公开数据集的基线实验中，我们希望在保持端到端推理效率的同时，进一步提升对复杂纹理与边缘细节的重建能力。基于此，本章在U-Net的基础上提出Deep Leaky U-Net（DLU-Net），并用于本章公开数据集重建实验。该网络结构也在作者的科研论文中出现（见\texttt{ref/个人科研成果/论文.pdf}），后续第\ref{chap:method}章将以此为基础展开面向自建实验系统的数据集与方法设计。

\subsection{DLU-Net结构概述}\label{sec:dlunet_detail}

DLU-Net作为U-Net的改进版本，核心思想是通过\textbf{更深的特征提取路径（Deep）}与\textbf{LeakyReLU激活（Leaky）}增强网络表达能力并缓解梯度传播问题。其具体结构以你在工程实现与科研论文中的定义为准，本章给出概念性描述与复现要点：
\begin{enumerate}
    \item \textbf{激活函数替换}：将卷积块中的ReLU替换为LeakyReLU，以降低“神经元死亡”风险。
    \item \textbf{网络深度/宽度调整（可选）}：通过增加下采样层级或通道数提升表达能力（需在论文中给出明确配置）。
    \item \textbf{训练稳定性增强（可选）}：如批归一化/残差连接/正则化等（若使用，请在实现小节中说明）。
\end{enumerate}

\section{训练策略与评价指标}\label{sec:train_metric}

\subsection{损失函数}\label{sec:loss}

本章以像素级回归为主，采用MAE（L1）损失作为基础损失：
\begin{equation}
\mathcal{L}_{\text{MAE}} = \frac{1}{N}\sum_{i=1}^{N}|\hat{Y}_i - Y_i|.
\end{equation}
在后续实验中也可按需要加入SSIM损失或频域约束等复合损失（第\ref{chap:method}章将结合自建数据进一步讨论）。

\subsection{优化器与学习率策略}\label{sec:optimizer}

优化器采用Adam，并参考文献\citep{sun2024ol_calibrationfree}的设置：初始学习率为$1\times 10^{-4}$，并按固定步长衰减；批大小设置为64。为保证公平对比，U-Net与DLU-Net采用一致的优化策略与训练轮数。

\subsection{评价指标}\label{sec:metrics}

为全面评估重建质量，本章采用以下指标：
\begin{itemize}
    \item \textbf{相关系数（Correlation Coefficient, CC）}：衡量重建图像与目标图像的相似性。文献\citep{sun2024ol_calibrationfree}使用二维相关系数作为主要“保真度（fidelity）”指标。
    \item \textbf{峰值信噪比（PSNR）}与\textbf{结构相似度（SSIM）}：常用于图像重建质量评估。
    \item \textbf{推理时间/帧率}：用于评估实时性潜力（尤其适用于内窥成像场景）。
\end{itemize}

\subsection{与参考文献结果的关系}\label{sec:compare_paper}

文献\citep{sun2024ol_calibrationfree}报告其端到端网络在实验数据上可实现毫秒级重建，并取得较高保真度。由于本章的网络结构（DLU-Net）与其网络实现可能不同，且数据集划分、预处理方式等细节会影响结果，因此本章更强调“复现实验流程与相对对比结论”。在后续章节中，我们将把这些结论迁移到自建实验系统，并进一步评估对光纤扰动、噪声以及跨域泛化的影响。

\section{结论性观察与对后续工作的启示}\label{sec:insights_to_next}

通过公开数据集的基线复现与DLU-Net改进实验，本章形成以下结论性观察（建议你结合实验结果补充具体数据支撑）：
\begin{enumerate}
    \item \textbf{端到端网络能够有效学习散斑到目标图像的映射}：在手写数字与服装两类数据上均能实现可辨识的重建结果。
    \item \textbf{数据复杂度提升会显著增加重建难度}：服装数据集的纹理与结构更复杂，模型更容易出现细节缺失或局部伪影。
    \item \textbf{DLU-Net在细节/鲁棒性上具有潜在优势}：相较U-Net，DLU-Net在某些样本上能更好地保持边缘与细节（需用定量指标或可视化对比支撑）。
    \item \textbf{训练与预处理细节对结果影响显著}：归一化方式、数据增强、学习率策略等都会影响收敛与泛化。
\end{enumerate}

基于上述观察，第\ref{chap:method}章将围绕“自建实验系统的数据集构建”和“面向实际光纤扰动的鲁棒重建”展开，进一步验证DLU-Net及其改进策略在真实实验场景中的有效性。

\section{本章小结}

本章基于文献\citep{sun2024ol_calibrationfree}公开的数据集建立了散斑图像重建的复现实验流程，并以U-Net为基线模型开展对比研究。在此基础上提出并验证了DLU-Net改进网络的可行性，给出了训练策略、评价指标与结果分析框架，并总结了若干结论性观察，为后续第\ref{chap:method}章的实验系统设计、网络结构选择与性能评估提供了依据与动机。

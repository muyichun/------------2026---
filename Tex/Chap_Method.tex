\section{双编码方案原理与实现}\label{sec:dual_encoding}

\subsection{编码方案概述}\label{sec:encoding_overview}

传统的多模光纤成像方法通常仅利用单一维度的信息编码（如空间位置或强度分布），限制了光纤的信息传输容量。为充分利用多模光纤支持的数千个空间模式，本文提出\textbf{双编码（Dual Encoding）}方案：同时利用\textbf{全息编码}和\textbf{偏振编码}，将目标图像信息嵌入到空间维度和偏振维度。

双编码方案的核心思想是：
\begin{enumerate}
    \item \textbf{全息编码}：利用计算全息方法，将目标图像调制到不同的空间位置（全息标签），使得不同图像在光纤输入端具有不同的角度谱分布。
    \item \textbf{偏振编码}：通过控制入射光的偏振态（水平偏振、垂直偏振或混合偏振），为每幅图像赋予额外的偏振标签。
\end{enumerate}

这种双编码策略使得散斑图案同时包含空间信息和偏振信息，显著增强了散斑模式的可区分性和信息容量。

\subsection{全息编码原理}\label{sec:holographic_encoding}

全息编码的目的是将目标图像调制到特定的空间位置，从而在多模光纤输入端产生不同的模式激发分布。本文采用\textbf{计算全息（Computer-Generated Holography, CGH）}方法实现全息编码。

\subsubsection{全息图生成}

给定目标图像$I(x,y)$和期望的全息坐标$(x_0, y_0)$，全息编码的目标是生成一个相位全息图$\phi(x,y)$，使得光场在傅里叶平面的$(x_0, y_0)$位置处再现目标图像。

在傅里叶全息的框架下，全息图可以通过添加线性相位梯度实现空间位移：
\begin{equation}
\phi(x,y) = \arg\{I(x,y)\} + 2\pi\left(\frac{x \cdot x_0}{\lambda f} + \frac{y \cdot y_0}{\lambda f}\right),
\end{equation}
其中，$\lambda$为光波长，$f$为傅里叶透镜的焦距，$\arg\{\cdot\}$表示相位提取操作。

\subsubsection{全息标签设计}

为增加散斑模式的多样性，本文设计了多个全息标签，对应不同的空间坐标$(x_0, y_0)$。\draftmark{待你确认：全息标签的具体坐标设置，如$(x_0, y_0) \in \{(-d, 0), (0, 0), (+d, 0)\}$或其他网格分布。}

不同的全息标签使得目标图像在光纤输入端激发不同的模式组合，从而产生可区分的散斑图案。

\subsection{偏振编码原理}\label{sec:polarization_encoding}

偏振编码利用光的偏振态作为额外的信息维度。多模光纤支持正交的两个偏振模式（TE模式和TM模式），通过控制入射光的偏振态，可以为图像信息赋予偏振标签。

\subsubsection{偏振态定义}

本文考虑三种偏振态：
\begin{itemize}
    \item \textbf{水平偏振（H-pol）}：电场矢量沿水平方向振动。
    \item \textbf{垂直偏振（V-pol）}：电场矢量沿垂直方向振动。
    \item \textbf{混合偏振（H+V）}：水平和垂直偏振分量同时存在。
\end{itemize}

偏振态的差异会导致光纤中不同偏振模式的激发和耦合，从而产生具有不同统计特性的散斑图案。

\subsubsection{偏振编码实现}

偏振编码通过光学元件实现偏振态的控制。具体而言，利用\textbf{偏振分束器（Beam Displacer, BD）}将不同偏振分量分离或合并，实现对入射光偏振态的精确调控。详细的光学系统设计见第\ref{sec:optical_setup}节。

\subsection{双编码的信息容量分析}\label{sec:capacity_analysis}

双编码方案显著提升了多模光纤的信息传输容量。假设全息编码提供$N_{\text{holo}}$个空间标签，偏振编码提供$N_{\text{pol}}$个偏振态，则理论上可实现$N_{\text{holo}} \times N_{\text{pol}}$路复用传输。

在本文的实验中，\draftmark{待你确认：全息标签数量$N_{\text{holo}}$和偏振态数量$N_{\text{pol}}$的具体值，如$N_{\text{holo}} = 9$，$N_{\text{pol}} = 3$。}

双编码产生的散斑图案具有更高的多样性和可区分性，为后续的深度学习重建和无监督聚类分析提供了丰富的特征空间。

\section{DeepLeakyU-Net网络架构}\label{sec:dlu_net}

\subsection{标准U-Net回顾}\label{sec:standard_unet}

如第\ref{chap:theory}章所述，U-Net是一种广泛应用于图像重建任务的编码器-解码器网络，具有以下特点：
\begin{itemize}
    \item 对称的U型结构，由收缩路径（编码器）和扩张路径（解码器）组成。
    \item 跳跃连接（Skip Connections）直接传递浅层特征到对应的深层，保留空间细节信息。
    \item 多尺度特征融合，兼顾全局语义和局部细节。
\end{itemize}

第\ref{chap:preliminary}章中使用的标准U-Net在公开数据集上取得了良好的重建效果。然而，在本文的双编码散斑重建任务中，散斑模式的复杂性和多样性提出了更高的要求。

\subsection{DeepLeakyU-Net的设计动机}\label{sec:dlu_motivation}

标准U-Net通常使用ReLU（Rectified Linear Unit）作为激活函数：
\begin{equation}
\text{ReLU}(x) = \max(0, x).
\end{equation}

ReLU的优点是计算简单、收敛快速，但其主要缺点是\textbf{死亡神经元（Dead Neurons）}问题：当输入为负值时，ReLU输出恒为零，导致梯度为零，神经元停止学习。

为克服这一问题，本文提出使用\textbf{Leaky ReLU}激活函数：
\begin{equation}
\text{Leaky ReLU}(x) = \begin{cases}
x, & x \geq 0, \\
\alpha x, & x < 0,
\end{cases}
\end{equation}
其中，$\alpha$为一个小的正数（通常取$\alpha = 0.01$或$0.2$）。

Leaky ReLU允许负值输入产生非零的小梯度，从而避免神经元死亡，改善梯度流动，提升网络的表达能力和训练稳定性。

基于此，本文提出\textbf{DeepLeakyU-Net (DLU-Net)}：在标准U-Net的基础上，将所有ReLU激活函数替换为Leaky ReLU，并适当增加网络深度和通道数，以增强特征提取能力。

编码器由多个卷积块和下采样层组成，逐步提取高层次特征并降低空间分辨率。每个卷积块包含：
\begin{itemize}
    \item 两个$3\times3$卷积层（padding=1，保持尺寸不变）
    \item Batch Normalization（批归一化）
    \item Leaky ReLU激活函数（$\alpha = 0.2$）
\end{itemize}

下采样通过$2\times2$最大池化（Max Pooling）实现，使特征图尺寸减半，通道数加倍。

\draftmark{待你确认并补充：编码器的具体层数、每层的通道数，如：}
\begin{itemize}
    \item 第1层：输入通道1（灰度散斑图），输出通道64
    \item 第2层：输入通道64，输出通道128
    \item 第3层：输入通道128，输出通道256
    \item 第4层：输入通道256，输出通道512
    \item 瓶颈层：输入通道512，输出通道1024
\end{itemize}

\subsubsection{解码器（Decoder）}

解码器由多个上采样层和卷积块组成，逐步恢复空间分辨率并融合编码器的跳跃连接特征。每个解码器模块包含：
\begin{itemize}
    \item $2\times2$转置卷积（Transpose Convolution）或上采样（Upsampling），使特征图尺寸加倍，通道数减半。
    \item 跳跃连接：将编码器对应层的特征图拼接（Concatenate）到上采样后的特征图。
    \item 两个$3\times3$卷积层（padding=1）
    \item Batch Normalization
    \item Leaky ReLU激活函数（$\alpha = 0.2$）
\end{itemize}

\subsubsection{输出层}

解码器最后一层通过$1\times1$卷积将特征图通道数映射到目标图像的通道数（对于灰度图像为1），并使用Sigmoid激活函数将输出限制在$[0, 1]$区间：
\begin{equation}
\hat{I} = \text{Sigmoid}(\text{Conv}_{1\times1}(\text{Decoder Output})).
\end{equation}

\subsection{损失函数设计}\label{sec:loss_function}

为同时优化重建图像的像素精度和结构相似性，本文采用\textbf{复合损失函数（Composite Loss）}：
\begin{equation}
\mathcal{L}_{\text{total}} = \lambda_1 \mathcal{L}_{\text{MAE}} + \lambda_2 \mathcal{L}_{\text{SSIM}},
\end{equation}
其中：
\begin{itemize}
    \item $\mathcal{L}_{\text{MAE}}$为平均绝对误差（L1损失），衡量像素级误差：
    \begin{equation}
    \mathcal{L}_{\text{MAE}} = \frac{1}{N}\sum_{i=1}^{N}|\hat{I}_i - I_i|.
    \end{equation}
    \item $\mathcal{L}_{\text{SSIM}}$为结构相似度损失，衡量结构信息的一致性：
    \begin{equation}
    \mathcal{L}_{\text{SSIM}} = 1 - \text{SSIM}(\hat{I}, I).
    \end{equation}
    \item $\lambda_1$和$\lambda_2$为权重系数，用于平衡两项损失的贡献。\draftmark{待你补充：$\lambda_1$和$\lambda_2$的具体取值，如$\lambda_1 = 1.0$，$\lambda_2 = 0.5$。}
\end{itemize}

\subsection{优化器与学习率策略}\label{sec:optimizer_lr}

\subsection{聚类质量定量评估}\label{sec:clustering_metrics}

为定量评估聚类效果，本文采用以下指标：
\begin{itemize}

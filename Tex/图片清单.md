## 注意事项

1. **图片格式**: 建议使用PNG格式（无损压缩），分辨率至少300 DPI
2. **图片尺寸**: 大图建议宽度至少3000像素，确保打印清晰
3. **颜色方案**: 
   - 聚类图使用区分度高的颜色映射（如tab20、viridis等）
   - 偏振态统一使用：红色（s偏振）、绿色（p偏振）、蓝色（混合偏振）
4. **字体大小**: 图中文字（坐标轴标签、图例等）应足够大，建议至少12pt
5. **图例位置**: 确保图例不遮挡重要数据
6. **子图标注**: 多子图的大图应清晰标注(a)、(b)、(c)等

## LaTeX插入图片的方法

在LaTeX中，图片已经预留了位置，只需取消注释并确保文件路径正确：

```latex
\begin{figure}[!htbp]
    \centering
    \includegraphics[width=0.95\textwidth]{Img/4/文件名.png}  % 取消这行的注释符号%
    \bicaption{中文标题}{English caption}
    \label{fig:标签名}
\end{figure}
```

例如，对于双编码原理图：
```latex
\includegraphics[width=0.90\textwidth]{Img/4/dual-encoding-principle.png}
```

只需将生成的图片文件放到 `Img/4/` 目录下，然后删除 `\includegraphics` 行前面的 `%` 注释符号即可。
